# Music Streaming Web App

A full-stack music streaming web application built with Spring Boot, Java, and frontend technologies. This app supports user authentication, playlist management, collaborative playlists, and secure audio streaming.

---

## Features

### User Authentication
- Sign up / Login
- Forgot password with OTP verification
- Google OAuth2 login

### Music Streaming
- Stream songs directly in the browser
- JWT-based secure streaming
- Supports pause, play, seek, and real-time buffering

### Playlists
- Create, Read, Update, Delete (CRUD) playlists
- Collaborative playlists: multiple users can contribute
- Join a collaborative playlist using a unique code provided by the owner

### Frontend
- Responsive design
- Interactive audio player with progress bar
- OTP and password reset modals
- Error & success popups for user feedback

---

## Technologies Used

- **Backend**: Spring Boot, Spring Security, Spring Data JPA, JWT
- **Database**: MySQL
- **Frontend**: HTML, CSS, JavaScript
